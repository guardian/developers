Open Source

It’s not just our journalism that’s open. Many of our projects are hosted on GitHub and we actively contribute to the open source community.

Active projects

Here are a few of our favourite open source projects. For a list of all publicly available repos, see the Guardian’s GitHub account.

  • Frontend

    Source for the Guardian’s next-generation responsive website.

  • Scribe

    A rich text editor framework for the web platform, with patches for browser inconsistencies and sensible defaults.

  • Riff Raff

    The Guardian’s Scala-based deployment library designed to help automate deploys.

  • gu-who

    A monitoring robot that sorts out who should be in your GitHub organisation and enforces best practices for account security.

  • source

    Components library for Source: the Guardian's Design System

  • prout

    A GitHub robot that monitors whether merged pull requests are in production or not. Makes sure your delivery pipeline is working and developers are thinking about code in production rather than in master.

Projects we've incubated

A number of projects that started at the Guardian are now truly open and a managed by their own community.

  • sass-mq

    A Sass mix-in that helps manipulating media queries in an elegant way.

  • Mockito

    A mocking library for Java with expressive verifications.
